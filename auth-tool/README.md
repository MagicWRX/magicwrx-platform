# Auth Tool

A standalone authentication service built with Next.js 15 and React 19.

## Features

- Modern Next.js 15 with App Router
- React 19 with TypeScript
- Ready for Firebase Authentication
- OAuth provider support
- Microservice architecture ready

## Getting Started

```bash
# Install dependencies
npm install

# Run the development server
npm run dev
```

Open [http://localhost:3001](http://localhost:3001) to view the application.

## Deployment

This auth tool is designed to be deployed as a separate Vercel project for microservice architecture.
